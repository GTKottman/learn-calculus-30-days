# Day file template (`views/days/day-NN.md`)

Copy this structure for each day. Replace `NN` with `01`–`30`. Checkpoint days **replace or extend** the Practice section per [`checkpoint-schedule.md`](checkpoint-schedule.md).

```markdown
# Day NN — Title

## Day objectives

- …

1. …
2. …

---

## Core concepts

… concise, rigorous prose …

## Figure N — Short title

**Takeaway:** One sentence.

1. …

```

## Checkpoint days (7, 14, 21, 28)

- Add a **Checkpoint test** section with **mixed problems** and **no solutions** adjacent to each prompt (solutions only in `<details>` blocks at the **end** of the file, or in a separate companion file).

## Conventions

- Use **LaTeX** inline: `\( ... \)` and display: `\[ ... \]` where needed.
- Every day must include **at least one labeled figure** (`## Figure …`) plus a **Takeaway** line.
